# NATHub 部署建议

## 1. 推荐环境

- Debian 12
- Python 3.11+
- SQLite（MVP）

---

## 2. 部署方式

推荐部署在：

- PVE 虚拟机
- 独立 x86 Linux 主机

不推荐部署在：

- OpenWrt / MT7621

---

## 3. 运行方式

开发环境：

- Flask 内置调试运行

生产环境建议：

- gunicorn
- nginx 反向代理
- systemd 托管

---

## 4. 配置建议

配置文件应包括：

- SECRET_KEY
- 数据库连接
- 凭据加密主密钥
- 日志目录
- 调试开关

---

## 5. 安全建议

- 控制器只暴露管理端口
- 使用 HTTPS
- 设备凭据加密存储
- 控制器仅允许管理网段访问
